# Self-hosted fonts

These font files are bundled with the Refugio del Sátiro lending app to satisfy
GDPR (no runtime requests to Google Fonts) and ship under their original
open-source licenses.

## Open Sans (`open-sans-variable.woff2`)

Designed by Steve Matteson. Licensed under the
[SIL Open Font License v1.1](https://openfontlicense.org/).
This is the Latin-subset, variable-axis weight file from
`https://fonts.gstatic.com/s/opensans/v44/...`.

## Oswald (`oswald-variable.woff2`)

Designed by Vernon Adams, Kalapi Gajjar, Cyreal. Licensed under the
[SIL Open Font License v1.1](https://openfontlicense.org/).
This is the Latin-subset, variable-axis weight file from
`https://fonts.gstatic.com/s/oswald/v57/...`.

The OFL allows bundling, modification, and redistribution provided the fonts
are not sold by themselves and the same license is preserved.
